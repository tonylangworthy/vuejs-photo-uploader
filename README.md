# Vue Photo Uploader
![Vue Photo Uploader](uploader.png)

A simple demo to load images, resize (current width 800px), and upload. 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
